# Absolute Two Face - MVP 需求文档

## 项目概述

**产品名称：** Absolute Two Face  
**定位：** AI 人脸风格化对比工具  
**目标用户：** 海外 C 端用户  
**核心价值：** 上传一张人脸照片，一键生成两种风格对比图

---

## 商业模式

**免费层：**
- 每日免费生成 1 次（吸引用户）

**付费层：**
- 单次生成：$2.99（高清下载）
- 月卡：$9.99（无限次生成）
- 年卡：$49.99（无限次生成）

---

## 技术架构

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Next.js + Tailwind CSS | 部署在 Cloudflare Pages |
| 后端 API | Cloudflare Workers | 处理上传、调用 AI |
| AI 服务 | Replicate API | InstantID / IP-Adapter FaceID |
| 图片存储 | Cloudflare R2 | 存储用户上传和生成的图片 |
| 支付 | Stripe | 信用卡支付 |
| 域名 | 用户自备 | 托管在 Cloudflare |

---

## MVP 功能清单（P0 必须有）

### 1. 首页
- [ ] 品牌展示（Logo、标语）
- [ ] 上传按钮（拖拽或点击上传）
- [ ] 示例展示（展示 2-3 组对比图效果）
- [ ] 简单说明（3 步：上传 → 生成 → 下载）

### 2. 图片上传
- [ ] 支持格式：JPG、PNG、WebP
- [ ] 文件大小限制：最大 10MB
- [ ] 人脸检测：上传后检测是否包含人脸
- [ ] 无人脸提示：未检测到人脸时提示用户换图

### 3. 风格选择
- [ ] 预设 2 种对比风格（MVP 阶段固定）：
  - **Style A：** Anime / 动漫风格
  - **Style B：** Cyberpunk / 赛博朋克风格
- [ ] 后续版本：支持用户自选风格组合

### 4. 生成过程
- [ ] 显示加载动画（预计等待时间：10-30 秒）
- [ ] 调用 Replicate API 生成 2 张风格化图片
- [ ] 失败重试机制（API 超时或失败时自动重试 1 次）

### 5. 结果展示
- [ ] 左右对比展示（原图 vs 风格 A vs 风格 B）
- [ ] 低分辨率预览（免费用户）
- [ ] 水印（免费用户）

### 6. 下载/付费
- [ ] 免费用户：带水印、低分辨率
- [ ] 付费用户：高清无水印下载
- [ ] Stripe 支付按钮（MVP 可先不做，用手动解锁）

### 7. 移动端适配
- [ ] 响应式设计，手机端可用

## 页面流程

```
首页 → 上传图片 → 检测人脸 → 生成中（等待） → 结果展示 → 付费下载
                                    ↓
                              失败 → 重试/报错
```
